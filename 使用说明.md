## ⚠️ 重要注意事项

### 发音功能
- **使用浏览器原生TTS**（不依赖外部API）
- 系统会自动选择最佳语音引擎（如Windows的Microsoft Aria）
- 如果觉得机械音太重，可以在Windows设置中安装更好的语音包

### 常见问题

#### 1. 页面一直转圈圈加载不出来
**原因：** JS文件有语法错误
**解决：** 已修复，确保使用最新版本

#### 2. 点击喇叭没声音
**原因：**
- 外部发音API（Google、有道）在中国网络环境下不稳定
- 连接超时或被CORB拦截

**解决：**
- 已改用浏览器原生TTS
- 刷新页面（Ctrl+F5）即可

#### 3. 加载失败，请刷新页面
**原因：** 直接双击HTML文件时，浏览器安全限制阻止加载JSON
**解决：** 使用本地服务器方式打开（http://localhost:8000）

#### 4. 图片加载失败
**原因：** 图片文件不存在
**解决：** 已添加自动回退机制，失败时显示emoji

## 🔧 技术细节

### 音频系统（audio.js）
- 使用 `window.speechSynthesis` API
- 自动加载并选择最佳语音引擎
- 优先选择美式英语真人音
- 语速：0.9，音调：1.0

### 生词本系统（storage.js）
- 使用 `localStorage` 保存数据
- 支持添加、删除、批量导入
- 数据格式：`{ en, cn, ipa }`

### 点击添加功能（app.js）
- 词汇单元格添加 `onclick` 事件
- 调用 `app.addToVocab(en, cn, ipa)`
- 自动去重，显示提示
