🚀 GitHub Pages 部署指南

将你的英语学习系统部署到云端，随时随地访问

✨ 完成后你将获得：
• 一个永久的网址（如：https://你的用户名.github.io/english-learning/）
• 可以在任何设备上访问
• 添加到手机主屏幕，像APP一样使用
• 完全免费，无需服务器

📋 准备工作

  • 一个 GitHub 账号（没有的话去 github.com 注册）
  • 这个文件夹里的所有文件
  • 10分钟时间

🎯 部署步骤（网页操作，最简单）

1创建 GitHub 仓库

  1. 登录 GitHub
  2. 点击右上角的 + → New repository
  3. 填写信息：
    • Repository name: english-learning （可以改成你喜欢的）
    • Description: 医疗英语学习系统
    • 选择 Public （公开）
    • ✅ 勾选 Add a README file
  4. 点击 Create repository
📸 截图位置：创建仓库页面

2上传文件

  1. 在仓库页面，点击 Add file → Upload files
  2. 把这些文件拖进去：
    • index.html
    • manifest.json
    • sw.js
    • css/ 文件夹
    • js/ 文件夹
    • data/ 文件夹
    • README_GITHUB.md （上传后改名为 README.md）
  3. 在底部填写：Initial commit
  4. 点击 Commit changes
⚠️ 不要上传：DEPLOY.md、启动.bat、test.html、standalone.html

3启用 GitHub Pages

  1. 在仓库页面，点击 Settings（设置）
  2. 左侧菜单找到 Pages
  3. 在 Source 下拉菜单选择：
    • Branch: main
    • Folder: / (root)
  4. 点击 Save
  5. 等待 1-2 分钟，页面顶部会显示： 
    Your site is live at https://你的用户名.github.io/english-learning/
📸 截图位置：Settings → Pages 页面

4访问你的网站

  1. 复制上面的网址
  2. 在浏览器打开
  3. 🎉 完成！
✅ 现在你可以把这个网址分享给任何人，或者保存到手机主屏幕！

📱 添加到手机主屏幕

iPhone (Safari)

  1. 用 Safari 打开你的网站
  2. 点击底部的"分享"按钮（方框带箭头 ⎋）
  3. 向下滚动，找到"添加到主屏幕"
  4. 点击"添加"
  5. ✨ 桌面上会出现一个图标 🏥

Android (Chrome)

  1. 用 Chrome 打开你的网站
  2. 点击右上角的三个点 ⋮
  3. 选择"添加到主屏幕"或"安装应用"
  4. 点击"添加"
  5. ✨ 桌面上会出现一个图标 🏥

🔄 以后如何更新内容

添加新的外教课程

  1. 进入你的 GitHub 仓库
  2. 点击 data/tutor-lessons.json
  3. 点击右上角的铅笔图标 ✏️（编辑）
  4. 在 "lessons" 数组里添加新课程数据
  5. 点击 Commit changes
  6. 等待 1-2 分钟，网站自动更新

❓ 常见问题

Q: 网站打不开？

A: 等待 2-5 分钟，GitHub Pages 需要时间部署。刷新页面查看部署状态。

Q: 修改后没生效？

A: 清除浏览器缓存（Ctrl+Shift+Delete），或者用无痕模式打开。

Q: 数据安全吗？

A: 仓库是公开的，任何人都能看到。建议：

  • 不要在数据里放真实姓名、公司名等敏感信息
  • 或者购买 GitHub Pro（$4/月）使用私有仓库

Q: 可以改网址吗？

A: 可以！在 Settings → Pages 可以设置自定义域名（需要购买域名）。

💡 提示：如果遇到问题，可以查看 GitHub 的官方文档，或者把错误信息发给我。
