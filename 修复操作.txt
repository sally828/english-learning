## 第一步：找到音频函数

访问网站：https://sally828.github.io/english-learning/

按 F12 打开「开发者工具」→「Elements」→ 搜索 `function speak` 或 `<script`。

找到 `<script>` 标签内的 `speak()` 函数定义。

---

## 第二步：修改音频逻辑

### 原代码：
```javascript
// 统一播放入口：Google真人音 → 有道备用 → 浏览器兜底
async function speak(text) {
  let ok = await googleSpeak(text);  // 中国不能用
  if (!ok) ok = await ydSpeak(text); // 被浏览器拦截
  if (!ok) ok = await synthSpeak(text);
  return ok;
}
```

### 修改后：
```javascript
// 统一播放入口：浏览器优先 → Google → 有道
async function speak(text) {
  let ok = await synthSpeak(text);  // 浏览器原生秒播
  if (!ok) ok = await googleSpeak(text);
  if (!ok) ok = await ydSpeak(text);
  return ok;
}
```
